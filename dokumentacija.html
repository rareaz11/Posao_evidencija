OPIS PROJEKTNOG ZADATKA

Zadatak je trazio slijedeće: Bilo je potrebno kreirati program u web razvojnom okruženju koji služi za evidenciju i pretragu ponuda poslodavaca i da korisnici mogu vidjeti svoje prijave. Koristeći zadani izvor podataka potrebno je dodavati, prikazati, izmjenjivati i brisati podatke. Program treba imati slijedeće funkcionalnosti: CRUD operacije nad određenim tablicama. Učitavanje podataka bez osvježavanja stranice prema zadanim kolonama kako je u projektu defnirano: Naziv Tvrtke, Kratak opis, Stručna sprema, Tip Ugovora, Kategorija posla.

OPIS PROJEKTNOG RJEŠENJA

Program je kreiran u alatu: Visual Studio Code. Ovaj program za izvor podataka koristi bazu podatka(PhpMyAdmin). Kreirana je baza podatka imena:"EVIDENCIJA_POSLOVA", koja ima slijdeće tablice:blokiraj,dogadaj,jobs,korisnici,prijavaposao,stranice. Za provođenje CRUD oprecija koristen je PHP kod, koji je provodio izmjene, dodavanja ,brisanje. Program ima tri vrste korisnika: Gost,Regirstriarni korisnik i Administrator. za učitvanje podataka bez osvježavanja stranice korišten je AJAX.
